# Day 02 - Linux Permissions

## Commands Learned

- ls -l
- chmod
- whoami
- id
- groups
- sudo

## Permission Numbers

- 777 = Full access
- 755 = Owner full, others read/execute
- 644 = Normal file permission

## What I Practiced

Today I learned Linux permissions and user management.

## Important Learning

Linux uses read, write and execute permissions for security.
